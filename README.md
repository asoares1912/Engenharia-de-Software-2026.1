# Engenharia de Software 2026.1

## 🎯 Objetivo do Semestre

Este repositório reúne todas as disciplinas cursadas no semestre 2026.1 da graduação em **Engenharia de Software**.  
O foco é consolidar conhecimentos em **arquitetura, modelagem, dados e controle de configuração**, aplicando boas práticas de desenvolvimento e versionamento de código.

## 🧩 Competências Desenvolvidas
- Aplicação de **padrões arquiteturais** e boas práticas de design.  
- **Modelagem UML** para documentação e comunicação técnica.  
- **Gerência de configuração** e versionamento com Git/GitHub.  
- Manipulação e análise de dados com **Python e Big Data**.  
- Implementação e consulta em **bancos NoSQL**.

## 🧠 Ferramentas Utilizadas

- Git e GitHub

- MongoDB, Redis
